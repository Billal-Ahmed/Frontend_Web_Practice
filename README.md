# Frontend Web Development Learning Repository

This repository is a collection of web development learning materials, practice files, and small frontend projects created for teaching students the fundamentals of frontend development.

## What This Repo Contains

This repo includes:

- HTML and CSS practice files for learning structure, styling, layout, and responsive design
- JavaScript examples and exercises covering basics such as variables, strings, loops, and scripting
- Mini frontend projects to apply concepts in real examples
- Notes and learning resources for HTML, CSS, and JavaScript

## Folder Overview

- HTML and CSS: basic HTML pages, CSS examples, layout practice, and styling concepts
- HTML + CSS + JS: JavaScript lessons, examples, and simple interactive files
- Mini projects: small projects like calculators, todo apps, digital clocks, API-based examples, and UI practice
- Notes: organized notes and study material for web development topics

## Topics Covered

- HTML basics and structure
- CSS styling and layout
- Flexbox and responsive design
- JavaScript fundamentals
- DOM manipulation and interactivity
- Working with APIs using fetch
- Simple frontend project building

## How to Use

1. Open any HTML file in a browser to view the example or project.
2. Use the notes folder as a reference while learning.
3. Explore the mini projects to see how concepts are applied in practice.

## Purpose

This repository is designed to help students learn and practice frontend web development in a simple, structured, and beginner-friendly way.
